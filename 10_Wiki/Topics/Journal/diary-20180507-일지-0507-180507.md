 #

인지

: 개인이 정보를 습득하거나 사용하는 과정

지각, 사고, 기억, 추론 등을 포함한다 (Stayor, 2010).

  

인지 기능은 해석 능력과 행동 역량에 비례한다.

  

#

인지적 왜곡

: 부정적으로 편향 된 사고처리의 유형

  

1.  과잉 일반화 (Overgeneralization)
    
2.  이분법적 사고 (Dichotomous thinking)
    
3.  선택적 추상화 (Selective abstraction)
    
4.  긍정의 격하 (Disqualifying the positive)
    
5.  과장 / 축소 (Maximization / Minimization)
    

  

-   갖춤새 효과 (Set effect) 
    

           : 이전의 학습 경험에 의한 사고의 고정

  

인지적 왜곡은 생각을 경직 시키거나 왜곡 시키며 왜곡 된 생각이 절대적이고, 자신과 세계에 대한 기본적인 믿음이 되도록 만든다 (neenan & dryden)

  

#

이분법적 사고 (Dichotomous thinking)

  

성공과 실패, 선과 악, 나쁜것과 좋은것 ..등 다양한 해석을 제한하는 사고 방식

  

1.  종교적 원리주의 : '나의 종교 이외는 나쁘다'
    
2.  완벽주의 : '완전하지 못하면 실패한 것'
    

  

이분법적 사고는 자존감에 영향을 미친다.

  

#

대한민국 사회

  

대한민국 사회 전반에는 이분법적 사고가 짙게 나타난다.

  

나름의 판단을 해보자면

1.  사회 전반에 정보력이 부족하다.
    
2.  사회 구성원들의 사고 능력의 미달
    
3.  사회 기억(기록)의 부재 및 왜곡
    
4.  지각 능력 및 기준 모호
    

  

그렇다면 어떤 처방이 필요한가?

1.  정보 전달 매체인 언론의 다양화
    
2.  문화 수준 향상
    
3.  역사 의식 강화
    
4.  민족 도덕성 강화
    

  

이렇게 보니 정말 부족함이 많은 사회다.

이를 극복하기 위한 개인들의 노력들이 쌓여가며 좋은 공동체 사회를 이루길 바래본다!
